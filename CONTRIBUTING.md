# Contributing to Free AI Deals & Promos 🤝

First off — thank you! This list stays valuable because people like you keep it fresh.

## ✅ What qualifies?

A submission is welcome if it fits **any** of these:

- 🆓 A **free tier** of an AI product (chatbots, coding tools, image/video/audio generators, APIs…)
- 💳 **Free credits** for an AI/LLM API or cloud AI platform
- 🎓 **Student / educator / nonprofit** discounts or free plans on AI tools
- 🚀 **Startup programs** that grant AI/cloud credits
- ⏳ **Limited-time promos** (e.g. "Pro free for 3 months with code X")

## 🚫 What does NOT qualify?

- Piracy, account sharing, or anything violating a product's Terms of Service
- "Free" trials that require a credit card AND auto-convert to paid without clear warning
  (you may still list it, but flag it clearly with ⚠️)
- Referral-link spam — referr links are only allowed if the *referree* gets a real bonus
- Paid products with no meaningful free offering

## 📝 How to add a deal

1. **Fork** the repo and create a branch: `add/<tool-name>`
2. Add your entry to the correct table in `README.md` (alphabetical order within each table)
3. Use this exact row format:

   ```markdown
   | [Tool Name](https://tool-link.com) | What you get free | Limits / requirements | ⏳ or 🔄 or ♾️ |
   ```

4. Keep descriptions **short and factual** — no marketing speak, no "best tool ever!!"
5. If it's a **limited-time promo**, include the expiry date if known: `(ends 2026-09-01)`
6. Open a **Pull Request** with:
   - The deal name in the title: `Add: ToolName — free 1M tokens`
   - A link to the official page proving the deal (pricing page, blog post, etc.)

## 🏷️ Legend refresher

| Symbol | Meaning |
|--------|---------|
| ♾️ | Free forever (open source / permanent free tier) |
| 🔄 | Recurring free allowance (daily/monthly credits) |
| ⏳ | Limited-time promo or one-time credit grant |
| 🎓 | Requires student/educator verification |
| 💼 | Requires startup/program eligibility |

## 🔍 Quality bar

- **Link to the official source** — no blog roundups or affiliate pages
- **Verify it yourself** before submitting — dead deals get removed
- One deal per row; don't bundle "Tool X + also check Tool Y"
- Spotted an expired deal? PRs removing stale entries are just as valuable! 🧹

## 💡 Suggesting without a PR

Open an issue using the **"Submit a Deal"** template and a maintainer will add it.

Thanks for keeping the AI community's wallet happy. 💸
